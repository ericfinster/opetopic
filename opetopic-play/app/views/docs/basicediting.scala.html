Basic Editing

In this section, we describe how to build opetopes graphically using the editor interface which you will find in all the applications on this site.

In the window below is an editor instance which will all you to build an opetope graphically.

Extrusion

The most basic technique is call extrusion and consists in enclosing a subtree of cells in a new cell.

Drop Extrusion

The opetopic version of degenerate cells are called drops. The editor has a separate facility for inserting drops.

Prepending a Glob

While the previous two operations work by inserting new space on the target side, it is sometimes convenient to insert a new cell on the source side. We can do that by selecting an external cell and asking the editor to insert a glob with the chosen cell as its target.
