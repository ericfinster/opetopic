Opetopic Categories

We will say that an opetopic set is an opetopic category if the following two conditions are satisfied:

Composites exist

Every pasting diagram admits a target universal filler. That is, given x, y, z as pictured, we obtain w and u such that u is target universal.

Coherence of Universals

For any target universal cell, if all but one face is also target universal, so is the remaining one.
